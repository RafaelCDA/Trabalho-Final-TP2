# Camada Core

A camada Core reúne componentes fundamentais utilizados na infraestrutura da aplicação, incluindo inicialização do banco, engine de conexão e gerenciamento de sessões.  
Esses elementos servem de base para as demais camadas, como repositórios, serviços e rotas.
